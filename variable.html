JavaScript variables are containers for storing data values.

In this example, x, y, and z, are variables, declared with the var keyword:

var x = 5; var y = 6; var z = x + y;

local var = Variables declared within a JavaScript function, become LOCAL to the function. Local variables have Function scope: They can only be accessed from within the function.

global var = A variable declared outside a function, becomes GLOBAL. A global variable has global scope: All scripts and functions on a web page can access it.
